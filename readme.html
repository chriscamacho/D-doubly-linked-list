    dList (doubly linked list)

    The class is a template meaning that all nodes contain the same type of data pointers

    The list can be iterated forward and backwards it can be sorted using a quick sort algorith the sort must be given a callback function so it can compare the "value" of your data type. nodes can be added, inserted and deleted searching for a node containing a particular pointer is by brute force callbacks can optionally be used for iterating
